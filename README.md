# expLODe

### v0 feature
- [x] Blender 4.2.0+ support
- [x] FBX import (y = up, z+ = forward)
- [x] FBX export (y = up, z+ = forward)
- [x] Smart UV Unwrap (island = 0.1, deg=66?)
- [x] LOD lvl 1 (Planar Decimate Modifier with 10 deg)
- [x] LOD lvl 2 (Unsubdivide 5 interations)
